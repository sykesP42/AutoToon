# AutoToon Skybox 版本增强设计文档

版本：1.0
日期：2026-06-04
状态：已批准

---

## 背景

用户反馈当前 `ui_skybox.py` 存在以下问题：
1. UI 反馈不明显 - 切换形状时按钮没有高亮显示
2. 自动旋转功能无效
3. 视角控制太弱 - 需要能旋转摄像机 XYZ 轴方便查看物体

本次增强旨在解决核心问题，并扩展功能以提升用户体验和渲染质量。

---

## 设计目标

### 核心问题修复

#### 1. UI 按钮高亮反馈
- 切换形状时，选中按钮显示高亮颜色（亮蓝色 `#4682B4`），未选中显示暗灰色 `#323235`
- 切换 Skybox 时同样高亮显示当前选项
- 实现方式：每次切换时重新绑定所有按钮的 theme

#### 2. 自动旋转功能修复
- 检查 `run()` 循环中的自动旋转逻辑是否正确执行
- 确保光源方向随 `rotation_angle` 变化
- 添加光源方向指示器（箭头图标）显示当前光源位置

#### 3. 摄像机控制系统（重点）
- **鼠标控制**：
  - 左键拖拽：旋转视角（Orbit）
  - 滚轮：缩放距离
  - 右键：平移视角
- **滑块控制**：
  - X 轴旋转角度滑块 (0° - 360°)
  - Y 轴旋转角度滑块 (0° - 360°)
  - Z 轴旋转角度滑块 (0° - 360°)
  - 缩放比例滑块 (0.5x - 2.0x)
- **同步机制**：鼠标操作实时同步到滑块显示

---

### 渲染质量改进

#### 4. 多面透视投影
- **立方体**：
  - 计算透视投影，显示前、顶、右三个可见面
  - 各面独立法线计算，正确光照响应
- **圆柱体**：
  - 显示侧面 + 顶面
  - 边缘过渡平滑处理

#### 5. 环境反射增强
- 增强菲涅尔效应，边缘区域更明显反射 Skybox
- 添加金属度参数控制反射强度
- 添加粗糙度参数控制反射模糊度

#### 6. 色阶平滑过渡
- 新增 `shade_smooth` 参数控制过渡宽度
- 使用 smoothstep 函数替代硬切
- 色阶数量保持可调 (2-4 级)

#### 7. 描边系统改进
- 支持描边颜色选择（黑/白/自定义 RGB）
- 支持描边宽度动态调整（已实现，需增强 UI）
- 支持内描边/外描边模式切换

---

### 新增形状

#### 8. 新形状类型
- **圆环 (Torus)**：环形几何体，展示材质在不同曲面的效果
- **圆锥 (Cone)**：锥形几何体，适合测试渐变效果
- **多面体 (Dodecahedron)**：多面体，测试复杂几何光照

---

### 多形状预览

#### 9. 同时预览多个形状
- 可选布局模式：
  - 单形状（当前默认）
  - 2x2 四形状网格
  - 1x3 三形状水平排列
- 同一材质参数应用到所有形状
- 便于对比不同几何体上的材质效果

---

### 自定义 Skybox

#### 10. 自定义 Skybox 图片
- 支持加载用户图片（PNG/JPG）作为 Skybox
- 自动适配尺寸到预览窗口
- 添加"加载 Skybox"按钮
- 保留预设 Skybox 选项

### 多面透视投影

立方体透视投影需要：
1. 定义相机位置（距离、角度）
2. 计算每个面的可见性
3. 对可见面进行投影变换
4. 计算边缘描边

### 鼠标交互

使用 Dear PyGui 的鼠标事件：
- `dpg.add_mouse_drag_handler(button=0)` - 左键拖拽
- `dpg.add_mouse_wheel_handler()` - 滚轮
- `dpg.add_mouse_drag_handler(button=1)` - 右键拖拽

## 开发顺序

按渲染质量优先顺序：

1. **Phase 1: 核心问题修复**
   - UI 按钮高亮
   - 自动旋转修复
   - 摄像机控制系统（鼠标+滑块）

2. **Phase 2: 渲染质量改进**
   - 多面透视投影
   - 环境反射增强
   - 色阶平滑过渡
   - 描边系统改进

3. **Phase 3: 新增形状**
   - 圆环、圆锥、多面体

4. **Phase 4: 多形状预览**
   - 布局切换
   - 同步渲染

5. **Phase 5: 自定义 Skybox**
   - 图片加载
   - 自定义背景

## 验收标准

1. 切换形状/Skybox 时按钮有明显高亮反馈
2. 自动旋转开关有效，光源方向可见变化
3. 鼠标拖拽可旋转视角，滑块同步显示角度
4. 立方体和圆柱体显示多个可见面
5. 边缘反射效果更明显
6. 色阶过渡平滑可调
7. 新形状渲染正确
8. 多形状布局工作正常
9. 自定义 Skybox 图片可加载显示
