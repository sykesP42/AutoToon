HybriToon 产品目标说明书

版本：MVP 1.0

更新日期：2026-05-28

文档用途：明确产品定位、核心价值、功能边界及技术策略，指导开发与评审。



一、产品定位

HybriToon 是一款 AI 辅助的实时风格化工具，为 Unreal Engine 5 用户提供“一键风格迁移”能力。

用户上传任意一张参考图（原画、照片、概念设计等），系统自动分析其视觉风格，并实时将当前 UE5 场景中的角色材质及后期效果同步为该风格，实现从“灵感”到“画面”的秒级转化。



目标用户：



游戏美术师（快速验证风格方向）



虚拟内容创作者（VTuber、动画短片）



独立游戏开发者（降低风格化调参门槛）



大创/毕业设计团队（展示 AI + 图形学融合能力）



二、解决的问题

痛点	HybriToon 解决方案

风格化调参复杂：MooaToon 材质有数十个参数，新手很难调出理想效果	AI 自动从参考图提取关键风格参数（阴影色、描边、高光等）

后期效果与材质分离：调完材质还需单独调节后处理体积	统一输出材质 + 后处理参数，全场景风格同步

反复试错耗时：每次换风格都需要手动重调所有滑块	一键切换，或通过预设库即时调用

缺乏直观对比	内置截图对比、实时预览窗、强度混合滑块

技术门槛高：普通用户无法在 UE5 中部署 AI 模型	采用“独立程序 + HTTP 通信”模式，推理完全在外部完成，UE5 仅接收参数

三、核心功能（MVP）

3.1 独立程序（AI 推理端）

参考图上传：支持 JPG/PNG，自动缩放至模型输入尺寸（224x224）。



AI 风格推理：加载预训练的 ONNX 模型，输出 N 维风格参数（N = 5 或 16，可配置）。



参数微调：所有输出参数以滑块形式展示，用户可手动修改并实时查看模拟预览（基于 OpenCV 的快速色调映射）。



一键发送：点击按钮，将当前参数通过 HTTP POST 发送至 UE5 插件。



风格预设管理：保存当前参数为 .style 文件，支持拖拽加载，方便复用。



连接状态监测：发送前自动 ping UE5 健康接口，若未连接则给出明确警告。



调试模式：输出完整请求/响应日志，便于排查问题。



暗黑主题 UI：专业工具感，适合长时间使用。



3.2 UE5 插件（渲染端）

本地 HTTP 服务：启动时监听 127.0.0.1:8080，提供两个接口：



GET /api/health：健康检查，返回 {"status":"ok"}。



POST /api/style：接收 JSON 格式参数 {"params": \[float]}。



参数解析与校验：验证数组长度、数值范围，非法请求返回详细错误信息。



材质参数应用：将前 3~5 个参数映射到选定模型的动态材质实例（阴影色 RGB、描边宽度、高光强度等）。



后处理参数应用：将剩余参数映射到场景主 PostProcessVolume 的饱和度、对比度、光晕强度、暗角、色温等属性。



重置功能：UI 按钮一键恢复材质至默认预设。



成功反馈：应用成功后播放短提示音，并返回 200 OK。



3.3 辅助工具

一键启动脚本：同时启动独立程序和 UE5 项目，演示环境即开即用。



独立程序打包：提供 .exe 版本，用户无需安装 Python 环境。



四、非功能需求

指标	要求

推理速度	独立程序从上传图片到输出参数 < 1s（CPU 上）

通信延迟	HTTP 往返 + 材质应用 < 50ms

插件兼容性	Unreal Engine 5.4 及以上

跨平台	Windows 10/11（后续可扩展 macOS）

即装即用	UE5 插件拷贝至 Plugins 目录即启用；独立程序解压即运行

错误处理	任何步骤失败均有明确提示，不静默崩溃

关键设计决策：



推理外置：避免在 UE5 中集成 ONNX，降低插件复杂度和兼容性问题。



HTTP 通信：标准协议，跨语言，易于调试和扩展（未来可支持 Blender、Maya 等）。



参数解耦：AI 只负责输出数值，具体映射到哪个材质参数由 UE5 插件内部决定，便于调整。

七、未来扩展方向（愿景）

模型扩展：支持更丰富的输出（法线扰动、细节纹理、动态元素等）。



实时联动：在独立程序中调节滑块，UE5 画面实时更新（WebSocket）。



批量处理：一次上传多张参考图，生成风格序列并导出。



云端风格库：社区分享风格文件，一键下载应用。



多软件适配：除 UE5 外，支持 Unity、Blender 等。



八、成功标准（MVP 验收）

用户上传任意一张图片，独立程序成功推理并输出参数。



点击“发送”，UE5 场景中选中的模型材质及后处理效果明显变化，与参考图风格神似。



关闭 UE5 后再次发送，独立程序提示“UE5 未连接”。



保存风格预设并重新加载，所有滑块恢复至保存值。



一键启动脚本能同时打开两个程序。



打包的独立程序在未安装 Python 的电脑上正常运行。



总结：HybriToon 不是一个“万能 AI”，而是一个 聚焦风格化渲染工作流的效率工具。它用最轻量的工程代价，实现了从“灵感”到“画面”的最短路径，让创作者专注于艺术表达，而非参数调试。
